## 程式使用方法
- 確保能執行Makefile的環境
- 在環境中使用 make all 來創建資料夾、編譯.c檔案
- 使用 make demo0、make demo1、make demo2、make demo3 來執行encoder、decoder，demo 0 1 2 3 分別對應到專案要求的 0 1 2 3
- 最後生成的檔案會放到對應的資料夾
